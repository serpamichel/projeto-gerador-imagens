Sobre o projeto:

O ger.IA tem como objetivo utilizar ferramentas de inteligência artificial para selecionar imagens aleatórias a partir das descrições inseridas.

Tecnologias utilizadas:

  • HTML5 para estruturação do conteúdo.
  • CSS3 para estilização.
  • Um modelo de IA para a seleção das imagens.
